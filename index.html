Columns must be placed inside a .row in order to function properly, though you can use them outside. It's best practice to ensure the number of cols per row does not exceed 12.

Examples of usage 
<!-- Good practice - Two equal columns, total cols <= 12 -->
<div class="row">
	<div class="col-6"></div>
	<div class="col-6"></div>
</div>

<!-- Bad practice - Total cols > 12 -->
<div class="row">
	<div class="col-6"></div>
	<div class="col-6"></div>

	<div class="col-7"></div>
	<div class="col-5"></div>
</div>

Classless CSS

This is a trend that has come about after the release of HTML5; specifically the new semantic tags available such as <header>, <details>, and <blockquote>. The idea behind it is that you can create a web page without the use of any CSS classes. This does of course have it's limitations and ultimately the sites that you can create with this are rather limited, however with this project I hope to broaden that range.

By using Simpl Grid, it does of course mean that your site will no longer be classless, but the focus of this project is entirely layout and utility based, so the styling will still be left completely to the main stylesheet.

I encourage everyone to try creating a classless site; it's a great tool for those learning about HTML and CSS, and it's good for professionals too as it challenges them to see what they can achieve with a limited toolset, and to learn about the new semantic tags. One of my favorite things is the ability to essentially change "themes" akin to that of Wordpress or Wix, as all you have to do is link a different classless sheet and your site now has a new design!
